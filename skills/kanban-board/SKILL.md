# Kanban Board

Use durable Kanban state for work that outlives one short fork/join call.

- `kanban_show` lists current tasks.
- `kanban_create` starts named work.
- `kanban_comment` records progress.
- `kanban_block` records a blocker and reason.
- `kanban_heartbeat` records liveness.
- `kanban_complete` closes finished work.
